# Телеграм-бот для открывания ворот

бот под raspberry pi, сигнал на ворота с gpio17

Сделан на заказ под конкретного пользователя

Возможность обновлений через git

Стабильнее чем предыдущий, нет ws-соединения для контроля
Удаленная перезагрузка только через самого бота

При подключении пользователя администраторы получают уведомления и могут сразу принимать решение

Есть возможность оставить заявку
